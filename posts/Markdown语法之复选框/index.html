Markdown语法之复选框

Feb 8 2021-02-08T13:17:59+08:00 by Yucapri
1 min

学习内容：Markdown复选框

在无序列表符号后面加上[]或者[x]代表选中或者未选中情况

1
2

- [x] ROW1  
- [ ] ROW2

或者

1
2

+ [x] ROW1  
+ [ ] ROW2

或者

1
2

* [x] ROW1  
* [ ] ROW2

效果如下：

  • ROW1
  • ROW2
